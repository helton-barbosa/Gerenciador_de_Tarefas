# Gerenciador_de_Tarefas
 Um simples gerenciador de tarefas utilizando a estrutura de dados lista em Python

#### **Exercício**: Criar um Gerenciador de Tarefas com Lista
##### Crie um programa que simule um gerenciador de tarefas. O programa deve permitir que o usuário realize várias operações em uma lista de tarefas, como adicionar, remover, marcar como concluída e visualizar as tarefas. O programa deve apresentar um menu com as seguintes opções:

1. Adicionar uma nova tarefaa
2. Marcar uma tarefa como concluída
3. Remover uma tarefa da lista
4. Visualizar todas as tarefas
5. Sair
